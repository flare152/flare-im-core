# Flare IM 聊天室客户端测试计划

## 测试目标
验证多个用户能够通过聊天室客户端正常发送和接收消息

## 测试环境
- 两个聊天室客户端实例：user1 和 user2
- 连接到同一聊天室（session_id: "chatroom"）
- 使用 WebSocket 协议连接到 Access Gateway

## 测试步骤

### 1. 连接测试
- [x] 启动 user1 客户端并成功连接
- [x] 启动 user2 客户端并成功连接
- [x] 验证两个客户端都显示已连接状态

### 2. 消息发送测试
- [x] 在 user1 客户端发送消息："Hello from user1!"
- [x] 验证 user2 客户端收到该消息
- [x] 在 user2 客户端发送消息："Hello from user2!"
- [x] 验证 user1 客户端收到该消息

### 3. 多条消息测试
- [x] user1 发送多条消息
- [x] user2 发送多条消息
- [x] 验证双方都能正确接收所有消息

### 4. 特殊字符测试
- [x] 发送包含中文的消息
- [ ] 发送包含表情符号的消息
- [ ] 发送较长的消息

## 预期结果
- 两个客户端都能成功连接到聊天室
- 消息能够在两个客户端之间双向传输
- 消息内容正确显示，无乱码
- 客户端能够正确区分自己发送的消息和他人发送的消息
